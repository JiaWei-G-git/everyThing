# 南网/电力行业需求访谈问题清单

## 使用说明
当项目描述中包含以下关键词时，使用此领域模板：南网、南方电网、计量、电能量、采集、台区、线损、供电局、班组、电能表、负荷、用电信息采集。

## 第一轮：业务背景与电网结构（问题 1-3）

1. **这个项目属于哪个电网层级？** 省公司、地市供电局、区县供电局、还是班组/供电所级别？
2. **系统的用户分布在哪些层级？** 比如省公司管理员、地市专责、区县运维、班组长、供电所抄表员？
3. **系统管理的设备范围是什么？** 省全域、地市范围、还是特定台区？

## 第二轮：核心功能与数据（问题 4-8）

4. **系统处理的核心数据是什么？**
   - 计量数据（电能表读数、负荷曲线、电量）？
   - 台区数据（台区档案、线损计算、拓扑关系）？
   - 采集数据（终端采集、通信状态、异常告警）？
   - 电能量数据（结算电量、交易电量、统计报表）？
5. **数据的来源和去向是什么？** 从用电信息采集系统获取？向营销系统推送？
6. **是否需要实时数据处理？** 比如实时负荷监测、实时告警推送？还是 T+1 日批处理即可？
7. **系统需要哪些核心功能模块？**
   - 数据采集与状态监测？
   - 线损计算与分析？
   - 异常预警与处理？
   - 电量统计与报表？
   - 设备档案管理？
   - 运维工单管理？
8. **是否有 KPI 考核指标？** 比如线损率目标、采集成功率目标、异常处理及时率？

## 第三轮：数据权限与组织层级（问题 9-12）【关键】

9. **数据权限如何按组织层级隔离？**
   - 省级用户：能看到全省数据？
   - 地市用户：只能看到本市数据？
   - 区县用户：只能看到本区县数据？
   - 班组/供电所用户：只能看到本班组/供电所数据？
10. **同一层级内不同角色权限是否有差异？** 比如地市管理员 vs 地市专责 vs 地市查询员？
11. **数据权限是否需要动态配置？** 比如临时授权某用户查看其他区域数据？
12. **敏感数据是否需要脱敏？** 比如用户电费、用户地址、手机号码？

## 第四轮：合规与技术约束（问题 13-16）

13. **是否有等保要求？** 等保几级？（电力行业通常为等保三级）
14. **是否有国产化要求？** 数据库是否需要达梦/人大金仓？中间件是否需要东方通？操作系统是否需要麒麟/统信？
15. **现有系统环境是什么？** 已有数据库类型（Oracle/MySQL/DWS/达梦）？现有中间件？网络环境（内网/隔离网）？
16. **是否需要与现有系统集成？** 比如与营销系统、GIS系统、用电信息采集系统对接？接口方式是什么？

## 第五轮：性能与扩展（问题 17-18）

17. **系统需要处理多大的数据量？**
   - 终端数量：几千？几万？几十万？
   - 数据频率：15分钟？1小时？1天？
   - 历史数据：保留多久？3年？5年？10年？
18. **是否有并发用户峰值？** 比如月初结算时集中查询？峰值并发多少？

## 第六轮：风险与优先级（问题 19-20）

19. **这个项目的关键时间节点是什么？** 比如迎峰度夏前上线？年底结算前完成？
20. **目前最大的技术风险或业务风险是什么？** 数据迁移？性能瓶颈？用户接受度？

## 南网特定术语速查
| 术语 | 含义 | 系统关联 |
|------|------|---------|
| 台区 | 变压器供电区域 | 台区档案、线损计算 |
| 计量点 | 电能计量位置 | 计量数据、结算电量 |
| 采集终端 | 用电信息采集设备 | 终端状态、通信监测 |
| 线损 | 线路损耗电量 | 线损计算、异常分析 |
| 负荷曲线 | 用电负荷变化曲线 | 实时监测、报表分析 |
| 供电局/班组 | 组织层级 | 数据权限隔离 |
| 专责 | 专业负责人 | 业务审核、数据管理 |
| 迎峰度夏 | 夏季用电高峰保障 | 系统稳定性要求 |
