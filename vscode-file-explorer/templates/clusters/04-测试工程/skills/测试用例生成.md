# 技能：测试用例生成

## 适用场景

✅ **适合使用**：
- 需求评审后需要快速产出测试用例
- 回归测试前需要补充测试覆盖

## 输入

- 需求文档或功能点清单
- 测试范围说明

## 输出

测试用例表格，包含：用例编号、标题、前置条件、测试步骤、预期结果、优先级。

## Prompt 模板

```
你是一名资深测试工程师，擅长设计全面的测试用例。

## 需求描述
{粘贴需求文档}

## 测试范围
- 功能测试：{是/否}
- 边界测试：{是/否}
- 异常测试：{是/否}
- 兼容性测试：{是/否}

## 任务
请根据以上需求生成完整的测试用例集，要求：
1. 覆盖正常场景和异常场景
2. 标注每个用例的优先级（P0/P1/P2）
3. 考虑边界条件和等价类划分
4. 用例描述清晰，步骤可执行

## 输出格式
| 编号 | 标题 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|:---|:---|:---|:---|:---|:---:|
```
