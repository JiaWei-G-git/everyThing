# 技能：根据需求生成原型

## 适用场景

✅ **适合使用**：
- 需求确定后需要快速产出页面原型
- 与客户确认页面布局前需要可视化方案

## 输入

- 需求文档或功能点清单
- 目标用户画像
- 设计约束（如移动端/桌面端）

## 输出

页面原型描述，包含：页面布局、组件列表、交互说明。

## Prompt 模板

```
你是一名资深UI设计师，擅长根据需求快速产出低保真原型。

## 需求描述
{粘贴需求文档}

## 目标用户
{描述目标用户}

## 设计约束
- 平台：{Web/移动端/桌面端}
- 设计风格：{简约/商务/活泼}

## 任务
请根据以上需求，设计页面原型方案，包含：
1. 页面布局（用 ASCII 或文字描述）
2. 关键组件列表
3. 交互流程说明
4. 需要特别注意的设计细节
```
