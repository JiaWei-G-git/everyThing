## 预留场景

| 场景 | 说明 |
|------|------|
| 文件系统 MCP | 让 AI 直接读写 Vault 文件，替代当前的手动 Shell 调用 |
| Git MCP | 自动执行 git add/commit/push，跟踪资产变更 |
| 数据库 MCP | 将会话元数据存入数据库，支持复杂查询和统计 |
| 搜索 MCP | 全文检索 Vault 内容，替代当前的 grep 扫描 |
