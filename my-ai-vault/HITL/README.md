# HITL 文件夹

本文件夹存放 AI 使用记录截图，用于黑客松评审。

## 需要放入的文件

### 1. Token 使用截图（必需）
- 时间范围：2026-04-25 10:00 ~ 2026-04-26 23:59
- 截图内容：模型名称、调用时间、Token 消耗量（input/output）
- 获取方式：
  - Kimi Code CLI: kimi usage 或网页版个人中心
  - Claude Code: 开发者控制台
  - Cursor: Usage 页面

### 2. AI 协作记录截图（建议）
- 展示与 AI 对话的关键节点
- 证明项目的 Vibe Coding 属性

请将所有截图放入此文件夹，命名格式：YYYY-MM-DD-描述.png
